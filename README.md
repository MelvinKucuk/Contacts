# Contacts

Simple app that reads contacts from .csv file embbed in the app. It displays all the users in a list, allow to tap to go to a detailed view where the user can edit and save the changes.

## Tech Stack
- Kotlin
- MVVM/MVI
- Jetpack Compose
- Room
- Hilt
- SplashScreen
- Compose Navigation
- Mockk
- JUnit
- SOLID
- Clean Architecture
- CSV Reader
